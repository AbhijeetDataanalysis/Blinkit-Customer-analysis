# Blinkit-Customer-analysis
🛒 Blinkit Customer Analysis | Power BI Project

This project focuses on analyzing Blinkit customer data to understand purchasing behavior, customer preferences, sales trends, and business performance. Using Power BI, the dashboard provides interactive visualizations and KPI tracking to uncover insights into customer segments, order patterns, revenue generation, and product performance. The analysis helps businesses improve customer retention, optimize inventory, and make data-driven decisions.

📌 Key Features
Customer Purchase Behavior Analysis
Sales & Revenue Performance Tracking
Customer Segmentation
Product Category Analysis
Order Trend Analysis
Interactive Filters and Slicers
KPI Dashboard for Business Insights
Data Cleaning and Transformation

📊 Key Insights
Top-selling product categories
Customer purchasing trends
Revenue and sales performance analysis
Customer retention and engagement metrics
Peak order periods and demand patterns
